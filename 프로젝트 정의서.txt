목표 스펙

주제 : 대통령 연설문 분석 for 대북갈등(군사적 도발 및 충돌, 민간피해)

데이터셋 범위 : 대통령실 홈페이지 내 대통령 연설문 전체

분석 계획 :
	- 군사적 도발 및 충돌 사건, 민간피해 사건 직전 1주일 내 연설문내 단어 빈도를 추출, 비교하여 가장 많이 사용된 단어 상위 10개를 선정.
	- 역대 연설문 전체의 상위 10개 빈도 단어와 백분율 비교.

1. 데이터 수집 
	- 대통령 연설문 웹크롤링
	
		- 입력 : 연설날짜
		- 출력 : 대통령 연설문 str
		
2. 데이터 정제(전처리)
	- 형태소 분석
	- 불용어 설정 및 제외

		- 입력 : str
		- 출력 : dataFrame 객체 (단어별 출현 빈도)
		
3. 데이터 시각화 
	- Word Cloud
	- 막대 그래프
	
		- 입력 : dataFrame 객체 (단어별 출현 빈도)
		- 출력 : 그래프
	
4. 백엔드
	- 스트림릿
