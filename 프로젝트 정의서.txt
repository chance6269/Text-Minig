보고자료

주제 : 대통령 연설문과 남북갈등(군사적 도발 및 충돌, 민간피해) 상관관계 분석

1. 배경 :
	
	- 주제 선정 이유 : 대통령 연설문 속 특정 단어들의 사용과 북한의 군사적 도발 및 충돌이 유의미한 상관 관계를 갖는지 알아보고자 한다.
	

2. 분석 계획 :

	- 수집 대상 데이터 선정 : 문민정부 이후 모든 연설문
	- 데이터 범위 설정 이유 : 
		1) 크롤링 함수 최적화 문제
			
		2) 시대적 상황 고려 : 문민정부 이전은 반공 정서로 인해 과격한 단어 사용이 잦아 결과의 신뢰성이 훼손될 수 있음.
		
	- 문민정부 이후 모든 연설문을 추출하여 수집
	- 군사적 도발 및 충돌 사건, 민간피해 사건 직전 1주일 내 모든 연설문 & 문민정부 이후 모든 연설문의 단어 빈도를 추출, 비교하여 가장 많이 사용된 단어 상위 10개를 선정.
	- 시각화하여 역대 연설문 전체의 상위 20개 빈도 단어와 비교.
	
	- 팀원 및 역할 담당
		- 성우진 : 데이터 수집
		- 오승필 : 데이터 전처리
		- 박주찬 : 데이터 시각화, PM
		- 이의재 : 서비스 구축(백엔드)
		
3. 분석 과정 :

	1) 데이터 수집

		- 대통령실 홈페이지 연설문 웹크롤링
		
		- 데이터셋 : 문민정부 이후의 모든 연설문
		
		- 툴 : request 패키지, BeautifulSoup 패키지
		
		- 웹크롤링 함수
			- 입력 : 연설문 게시판 내부 idn(artid)
			- 출력 : 대통령 연설문 str을 요소로 하는 리스트
		
		- 특이사항 :
			1. 목함지뢰 사건의 경우, 피해시점이 아니라 지뢰 매설 시점을 고려하여 수집할 필요가 있었다. 
		
		
	2) 데이터 정제(전처리)
	
		- 형태소 분석하여 명사 추출
		- 불용어 설정 및 제외
		
		- 툴 : 
			konlpy 패키지의 Komoran 모듈 : 한국어 형태소 분석기
			collections 패키지의 Counter 모듈 : 단어 빈도 카운팅
			re 모듈 : 불용어 처리
			
		- 전처리 함수
			- 입력 : 대통령 연설문 str 리스트
			- 출력 : dataFrame 객체 (단어별 출현 빈도)
		
		
	3) 데이터 시각화 
	
		- Word Cloud 생성 모듈 작성
		- 막대 그래프 생성 모듈 작성
			- 입력 : dataFrame 객체 (단어별 출현 빈도)
			- 출력 : 그래프 출력, 이미지 저장
			
		- 툴 : 
			matplotlib : 그래프 시각화, 막대 그래프 생성,
			wordcloud : 워드클라우드 제작 라이브러리
			PIL : (워드클라우드)이미지 처리 라이브러리
		
	
	4) 백엔드
		- 툴 : streamlit 패키지
	
4. 정리

	- 결론 :
		- 평균적으로 남북 충돌 사건 직전 연설문들 속 '평화'라는 단어 사용이 평시 연설문들보다 높은 빈도 순위를 보임.(평시 - 6위 / 사건 직전 - 1위)
		- 따라서, '평화'라는 단어의 사용이 잦을수록 남북 충돌 발생에 영향을 준다고 볼 수 있다.
		
	- 한계점 : 
		- 대남 도발의 원인에는 내부 정치적 이유나 경제적 압박, 국제 사회에 대한 주목도를 높이거나 국제적인 지원을 받기 위한 긴장 관계 유지, 내부결속 등 여러 가지 이유가 복합적으로 적용된다. 
		- 따라서 연설문 분석만으로 북한과의 대북 갈등 원인, 대남 도발 이유를 정확하게 파악하기는 어렵다.
