주제 : 대통령 연설문 분석 for 대북갈등[대남 도발](군사적 도발 및 충돌, 민간피해)

1. 배경 :

	- 주제
	- 주제 선정 이유 : [쉽게 배우는 R 텍스트 마이닝 책에서 대통령의 연설문을 불러오고 텍스트를 분석하는 내용을 학습한 뒤 미니 프로젝트에서 
	 파이썬을 사용해서 해당 내용을 응용해서 분석해보면 어떨까 라는 생각에 시작하게 되었습니다 과거부터 현재까지 대남 도발이 지속 중에 있는데
	 대남 도발 사건이 일어난 시점의 대통령의 전체 연설문과 대남 도발 1주일 전부터 전날까지의 연설문을 비교 했을때 주로 어떤 단어들을 많이 사용했는지 
	 단어 빈도수에 차이가 있는지에 대해 궁금한 점이 생겨서 대통령 연설문 분석 for 대북갈등[대남 도발](군사적 도발 및 충돌, 민간피해) 라는 주제를 선정하게 되었습니다]
	

2. 분석 계획 :

	- 수집 대상 데이터 선정 : 문민[국민의]정부 이후 모든 연설문
	- 문민[국민의] 정부 이후 모든 연설문을 추출하여 수집
	- 군사적 도발 및 충돌 사건, 민간피해 사건 직전 1주일 내 모든 연설문 & 문민[국민의] 정부 이후 모든 연설문의 단어 빈도를 추출, 비교하여 가장 많이 사용된 단어 상위 10개를 선정.
	- 시각화하여 역대 연설문 전체의 상위 10개 빈도 단어와 비교.
	
	- 역할 소개
		- 성우진 : 데이터 수집
		- 오승필 : 데이터 전처리
		- 박주찬 : 데이터 시각화, 프로젝트 기획
		- 이의재 : 서비스 구축(백엔드)
		
3. 분석 과정 :

	1) 데이터 수집 
		- 대통령실 홈페이지 연설문 웹크롤링
		
		- 데이터셋 : 문민[국민의]정부 이후의 모든 연설문
			- 목표 데이터셋 : 역대 대통령들의 모든 연설문
			- 축소 이유 : 
					1. 크롤링 함수 최적화 문제 :
					2. 시대적 상황의 문제 : 문민[국민의] 정부 이전은 반공 정서로 인해 과격한 단어 사용이 잦아 결과의 신뢰성이 훼손될 수 있음.
					[3. 시간 문제 : 데이터 수집 및 전치리 시간이 오래 걸리는데 비해 프로젝트 기간이 짧음]
		
		- 툴 : request 패키지, BeautifulSoup 패키지
		
		- 웹크롤링 함수
			- 입력 : 연설문 게시판 내부 idn(artid)
			- 출력 : 대통령 연설문 str이 저장된 리스트
		
		
	2) 데이터 정제(전처리)
	
		- 형태소 분석하여 명사 추출
		- 불용어 설정 및 제외
		
		- 툴 : 
			konlpy 패키지의 Komoran 모듈 : 한국어 형태소 분석기
			collections 패키지의 Counter 모듈 : 단어 빈도 카운팅
			re 모듈 : 불용어 처리
			
		- 전처리 함수
			- 입력 : 대통령 연설문 str이 저장된 excel 파일
			- 출력 : dataFrame 객체 (단어별 출현 빈도)
		
	3) 데이터 시각화 
	
		- Word Cloud 생성 모듈 작성
		- 막대 그래프 생성 모듈 작성
			- 입력 : dataFrame 객체 (단어별 출현 빈도)
			- 출력 : 그래프 출력, 이미지 저장
			
		- 툴 : 
			matplotlib : 그래프 시각화, 막대 그래프 생성,
			wordcloud : 워드클라우드 제작 라이브러리
			PIL : (워드클라우드)이미지 처리 라이브러리
		
	4) 백엔드
		- 툴 : streamlit 패키지
